## Creating a New ADR

1. Copy `0001-adrs-template.md` to `00XX-<title>.md`
2. Fill in the template
3. Update this index
4. Submit for review

## ADR Status Meanings

- **Proposed**: Under discussion
- **Accepted**: Approved for implementation
- **Deprecated**: No longer relevant
- **Superseded**: Replaced by another ADR
